Welcome to my portfolio website. Here I showcase three different projects from my three semesters at KEA. They highlight my progress in learning how to code, from 1.semester animated game to a 3.semester JavaScript based assignment.

3.Semester - KEA

Hacked Hogwarts Student List
A complex Javascript based assignment

In this assignment had to create an interface showing a list of students from which you can sort, filter as well as expel students, make them prefects or make them a part of an inquisitorial squad under certain conditions.

2.Semester - KEA - Exam project

Creating an online presence for a start-up
Develop and communicate a start-ups profile by creating a strong, professional online presence

In this assignment my group and I worked with a jewellry start-up called Ivaldesmykker - they had an existing homemade wordpress site, an empty instagram account and a facebook page that had just been initiated. There was no clear coherency in the way Ivaldesmykker expressed themselves online. So our task was to develop a Style & SoMe guide, a video(s) to be used in a fashion that would help this particular start-up promote themselves or their products. Finally we had to develop a new website and this is the part I played a role in.

1.Semester - KEA

Animated game
Create an interactive, animated game using only HTML, CSS & JavaScript

In this assignment we had to create and animated game, using the basics of browser technologies and also draw all graphics by ourselves and record and manipulte the sounds used in the game.
