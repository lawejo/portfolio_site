Hacked Hogwarts Student List

In this assignment I had to create an interface showing a list of students from which you can sort, filter as well as expel students, make them prefects or make them a part of an inquisitorial squad under certain conditions.

This assignment was large and complex enough that a full walkthrough would take too much time, instead I've highlighted some key points. Cleaning messy data

Before doing anything else we had to clean up the data we were given. On the top-right image you can see what we received and underneath that, what data I turned it into, to further work with it.

To get the clean data I created a function where I can loop through all the students and have the improved data returned.

Said function can be viewed in the embedded codepen on the right. In it I use a lot of array methods to ensure that even if a name has a capitalized letter in the middle of the name it is still returned with only the first letter capitalized.

It is from here that I create my new student object that I have created with all the relevant properties that are needed for this assignment

Filtering a list of objects

This I did by utilizing the object I created while cleaning up the data. You can filter by a student's house (Such as Gryffindor, Ravenclaw, Slythering or Hufflepuff) or if the student's are expelled. Therefore one of the filter functions looks as is displayed here on the left. If the object carries the value of "Ravenclaw" in this instance, it is returned true and then the accumulated list is used to display, only showing the students from Ravenclaw.

Comparing JSON data

A requirement of the assignment was to be able to determine a students "blood status" and to solve this I had to deal with fetching two different JSON files. One of these I showed a snippet from at the top of the page that delivered the students full name and house, and then this second one that has two arrays dividing students in to "Pure-blood" or "Half-blood" by their last name.

After fetching the second JSON file I created an object with two empty arrays inside called "half" & "pure" and pushed the students into these arrays accordingly. To the left you can see how my function looks. It's a function with another function nested inside.

In my first if-statement there are three possible outcomes that can be returned as "half", "full" or "muggle". What is returned is determined in the nested function that compares an array (which is part of the second JSON file turned into an array) and then the last name of the given student. If the student is nowhere to be found in either one they are returned as a muggle.
